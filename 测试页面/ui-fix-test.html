测试说明

这个页面故意禁用了DaisyUI和Tailwind CSS的加载，用来模拟CSS框架加载失败的情况。

预期效果：

  • 下拉菜单内容应该默认隐藏
  • 鼠标悬停时显示下拉内容
  • 页面布局应该保持整洁
  • 所有功能按钮都应该可点击

测试步骤：

  1. 检查页面加载时下拉菜单是否隐藏
  2. 将鼠标悬停在"主题切换"按钮上
  3. 将鼠标悬停在"帮助"按钮上
  4. 确认下拉内容正确显示和隐藏
